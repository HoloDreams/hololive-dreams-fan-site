# ニュース画像の手動更新

GitHub上で次の5枚を上書きすれば、ホームのニュース画像が変わります。

- code/news_img/01.jpg
- code/news_img/02.jpg
- code/news_img/03.jpg
- code/news_img/04.jpg
- code/news_img/05.jpg

リンク先を変える場合だけ、code/news_img/instagram-news.js の url を編集してください。
画像名は 01.jpg から 05.jpg のままにします。

おすすめサイズ: 横長 16:9、1280x720 または 960x540。
GitHubで上書き後、Pagesの公開処理が終わったら Ctrl+F5 で更新してください。
